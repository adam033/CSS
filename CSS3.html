Font Style

CSS dapat mengatur style dari teks pada halaman yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style text yaitu sebagai berikut.

  • Text-Align : Digunakan untuk pemerataan teks, dengan value right , center dan left
  • Color : Digunakan untuk memberikan warna pada teks
  • Letter-Spacing : Digunakan untuk mengatur jarak antar karakter
  • Word-Spacing : Digunakan untuk mengatur jarak antar karakter
